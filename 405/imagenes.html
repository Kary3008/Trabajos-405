¿Que es HTML5?


(HyperText Markup Language, versión 5) Es la quinta version del lenguaje HTML. Se encuentra en desarollo y en conjunto con CCS3, define los nuevos estandares de desarrollo web, rediseñando el codigo para resolver problemas y actualizándolo así a nuevas necesidades. Incorpora muchas características nuevas y proporciona una plataforma de desarrollo de complejas aplicaciones web (mediante los APIs).Nos permite una mayor interaccion entre nuestras paginas web y el contenido media (video, audio, entre otros) asi como unamayor facilidad a la hora de codificar nuestro diseño basico.
